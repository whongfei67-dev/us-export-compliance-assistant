# 美国出口合规助手 - 智能决策系统

基于美国出口管制法规 (EAR) 和 OFAC 制裁法规的智能合规检测与辅助决策系统。

## 产品概述

帮助进出口企业快速判断产品是否涉及美国出口管制合规风险，并提供专业的合规建议和文件生成服务。

## 核心功能

### 1. 合规检测
- 企业信息采集
- 买家/收货方信息审查
- 产品类别与ECCN编码识别
- 运输路线风险评估
- 全链路风险评分

### 2. 智能分析引擎
基于以下法规进行自动判断：
- **商务部 EAR** (Export Administration Regulations)
- **财政部 OFAC** 制裁名单
- **实体清单** (Entity List)
- **军事最终用途规定** (MEU)

### 3. 文件生成
- 出口许可证申请表 (BIS-748P)
- 最终用途声明书
- 情况说明函

## 适用场景

- 芯片/半导体出口检测
- 计算机设备出口检测
- 通信设备出口检测
- 传感器/雷达设备出口检测
- 任何涉及美国技术产品进出口

## 风险评估维度

| 维度 | 检查项 |
|------|--------|
| 买家国家 | 是否在制裁名单 |
| 买家类型 | 政府/军事/民用 |
| 产品类别 | ECCN编码敏感性 |
| 技术含量 | 美国技术比例 |
| 目的地 | 转运风险 |
| 最终用户 | 用途核实 |

## 合规等级说明

- **PROHIBITED (禁止)**: 交易被禁止，需立即停止
- **HIGH_RISK (高风险)**: 需要申请许可证
- **MEDIUM_RISK (中等风险)**: 建议谨慎处理
- **LOW_RISK (低风险)**: 基本可继续但需注意

## 重要声明

本系统仅供参考学习，实际合规判断请咨询专业出口管制律师。
